# 途灵 TripMind 智能旅行助手功能说明

## 1. 系统简介

途灵 TripMind 是一个基于 AI 的智能旅行助手系统，旨在为用户提供全方位的旅行规划、预订和管理服务。系统采用现代化的界面设计，结合先进的 AI 技术，为用户打造个性化的旅行体验。

## 2. 功能模块

### 2.1 用户管理

#### 2.1.1 注册
- **功能**：创建新用户账号
- **流程**：
  1. 填写邮箱、密码等基本信息
  2. 系统自动生成唯一的乘客 ID
  3. 密码加密存储
  4. 注册成功后自动登录

#### 2.1.2 登录
- **功能**：用户身份验证
- **流程**：
  1. 输入邮箱和密码
  2. 系统验证身份
  3. 生成 JWT token
  4. 登录成功后跳转到主界面

#### 2.1.3 个人中心
- **功能**：查看和管理个人信息
- **内容**：
  - 用户基本信息
  - 我的订单（机票、酒店、租车）
  - 退出登录

### 2.2 智能推荐

#### 2.2.1 当季推荐
- **功能**：根据当前季节推荐适合的目的地
- **算法**：基于季节匹配和热门程度
- **展示**：卡片式布局，显示目的地图片、名称、国家、标签等信息

#### 2.2.2 目的地筛选
- **功能**：根据地区、类型等筛选目的地
- **筛选条件**：
  - 地区：亚洲、欧洲、美洲、大洋洲
  - 类型：海岛、都市、自然
- **展示**：筛选结果以卡片形式展示

#### 2.2.3 一键规划
- **功能**：快速生成目的地行程
- **流程**：
  1. 点击目的地卡片上的"规划行程"按钮
  2. 系统自动生成行程建议
  3. 用户可以调整和确认行程

### 2.3 行程规划

#### 2.3.1 行程模板
- **功能**：提供预设的行程模板
- **模板类型**：
  - 城市深度游
  - 海岛度假
  - 自然风光
  - 文化探索

#### 2.3.2 自定义规划
- **功能**：根据用户需求定制行程
- **流程**：
  1. 输入目的地、天数、预算等信息
  2. 系统生成行程建议
  3. 用户调整行程细节
  4. 保存行程

#### 2.3.3 行程管理
- **功能**：查看和管理已保存的行程
- **操作**：
  - 查看行程详情
  - 修改行程
  - 分享行程
  - 删除行程

### 2.4 世界地图

#### 2.4.1 地图探索
- **功能**：交互式世界地图
- **操作**：
  - 缩放和拖动地图
  - 点击目的地查看详情
  - 查看目的地标记

#### 2.4.2 足迹记录
- **功能**：标记已访问和计划访问的地点
- **操作**：
  - 添加足迹
  - 编辑足迹
  - 删除足迹
  - 查看足迹统计

### 2.5 预算管理

#### 2.5.1 预算设置
- **功能**：设置旅行预算
- **内容**：
  - 总预算
  - 交通预算
  - 住宿预算
  - 餐饮预算
  - 其他预算

#### 2.5.2 支出跟踪
- **功能**：跟踪旅行支出
- **操作**：
  - 添加支出记录
  - 分类支出
  - 查看支出明细

#### 2.5.3 预算分析
- **功能**：分析预算使用情况
- **内容**：
  - 预算使用比例
  - 支出趋势
  - 预算建议

### 2.6 统计分析

#### 2.6.1 旅行数据
- **功能**：统计旅行相关数据
- **内容**：
  - 旅行次数
  - 旅行天数
  - 总花费
  - 访问国家数

#### 2.6.2 偏好分析
- **功能**：分析用户旅行偏好
- **内容**：
  - 喜欢的目的地类型
  - 出行季节偏好
  - 预算分配偏好

### 2.7 礼宾服务

#### 2.7.1 特殊请求
- **功能**：提交特殊旅行请求
- **内容**：
  - 特殊饮食要求
  - 特殊住宿需求
  - 其他特殊请求

#### 2.7.2 个性化服务
- **功能**：提供个性化旅行建议
- **内容**：
  - 基于用户偏好的推荐
  - 个性化行程建议
  - 当地体验推荐

### 2.8 智能聊天

#### 2.8.1 自然语言对话
- **功能**：与 AI 助手进行自然语言对话
- **能力**：
  - 回答旅行相关问题
  - 提供旅行建议
  - 处理旅行请求

#### 2.8.2 流式响应
- **功能**：实时显示 AI 回复
- **特点**：
  - 打字机效果

#### 2.8.3 工作流集成

- **流程**：
  1. 用户发送请求

  3. 调用相应工具
  4. 处理结果并回复用户

#### 2.8.4 审批流程
- **功能**：重要操作需要用户批准
- **操作**：
  - 显示操作详情
  - 用户批准或拒绝
  - 系统执行相应操作

### 2.9 通知系统

#### 2.9.1 订单通知

## 3. 使用指南

### 3.1 首次使用

1. **注册账号**：填写邮箱和密码，创建新账号
2. **登录系统**：使用注册的账号登录
3. **完善个人信息**：在个人中心完善个人信息
4. **开始规划**：使用智能推荐或行程规划功能开始规划旅行

### 3.2 旅行规划流程

1. **选择目的地**：从推荐或搜索中选择目的地
2. **制定行程**：使用行程模板或自定义规划行程
3. **预算管理**：设置旅行预算并跟踪支出
4. **预订服务**：预订机票、酒店、租车等服务
5. **行程管理**：查看和修改行程安排
6. **旅行记录**：记录旅行足迹和体验

### 3.3 智能助手使用

1. **发送请求**：在聊天界面输入旅行相关问题或请求
2. **查看回复**：实时查看 AI 助手的回复
3. **批准操作**：对需要批准的操作进行确认
4. **保存对话**：系统自动保存对话历史
5. **回顾历史**：在侧边栏查看历史对话

### 3.4 常见操作

#### 3.4.1 预订机票
1. 在聊天界面输入"预订机票"
2. 提供出发地、目的地、日期等信息
3. 系统搜索航班并显示结果
4. 选择航班并确认预订
5. 系统发送预订确认通知

#### 3.4.2 预订酒店
1. 在聊天界面输入"预订酒店"
2. 提供目的地、入住日期、入住人数等信息
3. 系统搜索酒店并显示结果
4. 选择酒店并确认预订
5. 系统发送预订确认通知

#### 3.4.3 规划行程
1. 在推荐视图选择目的地
2. 点击"规划行程"按钮
3. 系统生成行程建议
4. 调整行程细节
5. 保存行程

## 5. 常见问题

### 5.1 登录问题
- **问题**：忘记密码
  **解决**：点击登录页面的"忘记密码"链接，按照提示重置密码

- **问题**：账号被锁定
  **解决**：联系客服解锁账号

### 5.2 预订问题
- **问题**：预订失败
  **解决**：检查网络连接，或联系客服

- **问题**：修改预订
  **解决**：在个人中心的订单管理中修改

### 5.3 行程问题
- **问题**：行程冲突
  **解决**：调整行程安排，或咨询智能助手

- **问题**：行程取消
  **解决**：在行程管理中取消行程

### 5.4 技术问题
- **问题**：页面加载缓慢
  **解决**：检查网络连接，或刷新页面

- **问题**：功能无法使用
  **解决**：清除浏览器缓存，或联系客服
