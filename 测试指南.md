# 途灵 TripMind 测试指南

**访问地址：** https://tripmind.space

---

## 快速开始

注册账号或使用测试账号登录后，直接在对话框输入你的需求，AI 会自动理解并处理。

## 一、航班查询与预订

### 支持的城市（可互相查询航班）

| 出发城市 | 可到达城市 |
|---------|-----------|
| 北京（PEK）| 东京、巴黎、巴厘岛、瑞士、纽约、悉尼、迪拜、伦敦、曼谷、新加坡、马尔代夫、罗马、首尔、巴塞罗那、阿姆斯特丹、伊斯坦布尔、开罗、洛杉矶 |
| 上海（PVG）| 东京、巴黎、新加坡、马尔代夫、首尔、洛杉矶 |
| 广州（CAN）| 曼谷、新加坡 |
| 东京（NRT）| 北京、上海、巴黎、首尔 |
| 新加坡（SIN）| 北京、上海、巴厘岛、曼谷、悉尼、马尔代夫、首尔 |
| 迪拜（DXB）| 北京、伦敦、新加坡、开罗、伊斯坦布尔、马拉喀什 |
| 伦敦（LHR）| 北京、纽约、迪拜、阿姆斯特丹 |
| 纽约（JFK）| 北京、伦敦、洛杉矶、坎昆 |

### 测试话术示例

```
帮我查一下北京到东京的航班，下周出发
我想订一张上海飞新加坡的机票，后天出发，经济舱
查一下我的机票订单
帮我把机票改签到下下周
取消我的机票
```

> ⚠️ 敏感操作（预订、改签、退票）会弹出确认框，需要点击"批准"才会执行

## 注意事项

1. **敏感操作需确认**：预订、改签、退票等操作会弹出确认提示，需手动点击"批准"
2. **数据为测试数据**：所有订单均为模拟数据，不涉及真实交易
3. **AI 可能需要追问**：如果信息不完整，AI 会主动询问出发日期、目的地等信息
4. **历史对话**：左侧边栏可查看历史对话记录，点击可继续之前的对话
