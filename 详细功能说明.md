# 途灵 TripMind 智能旅行助手详细功能说明

## 1. 系统概述

途灵 TripMind 是一个基于 AI 的智能旅行助手系统，旨在为用户提供全方位的旅行规划、预订和管理服务。系统采用现代化的界面设计，结合先进的 AI 技术，为用户打造个性化的旅行体验。

### 1.1 系统架构

系统采用前后端分离架构：
- **前端**：React + Tailwind CSS + Vite
- **后端**：FastAPI + LangChain + MySQL
- **AI 引擎**：基于 LangChain 的状态图工作流

### 1.2 核心功能

- **智能旅行规划**：基于用户偏好和季节推荐目的地，生成个性化行程
- **一站式预订服务**：机票、酒店、租车等旅行服务的预订
- **预算管理**：旅行预算规划与跟踪
- **AI 智能助手**：基于大语言模型的旅行顾问
- **行程管理**：行程查看、修改、分享
- **数据统计分析**：旅行数据统计和偏好分析

## 2. 详细功能说明

### 2.1 用户管理模块

#### 2.1.1 注册功能

**功能描述**：允许新用户创建账号，获取系统访问权限。

**实现细节**：
- **前端**：
  - 注册表单：收集邮箱、密码、确认密码
  - 表单验证：实时验证邮箱格式、密码强度
  - 提交处理：显示加载状态，处理注册结果
  - 错误提示：显示注册失败原因

- **后端**：
  - 路由：`POST /api/register/`
  - 数据验证：使用 Pydantic 模型验证输入
  - 密码加密：使用 bcrypt 加密存储密码
  - 乘客 ID 生成：自动生成唯一的乘客 ID（格式：4位数字 + 空格 + 6位数字）
  - 数据库操作：将用户信息存储到数据库

**使用流程**：
1. 用户访问注册页面
2. 填写邮箱和密码
3. 点击注册按钮
4. 系统验证信息并创建账号
5. 注册成功后自动登录并跳转到主界面

#### 2.1.2 登录功能

**功能描述**：验证用户身份，授予系统访问权限。

**实现细节**：
- **前端**：
  - 登录表单：收集邮箱和密码
  - 记住我功能：使用 localStorage 存储登录状态
  - 提交处理：显示加载状态，处理登录结果
  - 错误提示：显示登录失败原因

- **后端**：
  - 路由：`POST /api/login/`
  - 身份验证：验证邮箱和密码
  - JWT 生成：生成 JSON Web Token
  - 响应：返回用户信息和 token

**使用流程**：
1. 用户访问登录页面
2. 填写邮箱和密码
3. 选择是否记住登录状态
4. 点击登录按钮
5. 系统验证身份并生成 token
6. 登录成功后跳转到主界面

#### 2.1.3 个人中心

**功能描述**：查看和管理个人信息，查看订单状态。

**实现细节**：
- **前端**：

## 4. 部署与维护

### 4.1 部署环境

- **前端**：
  - Node.js 18+
  - npm 9+
  - 静态文件服务器（如 Nginx、Apache）

- **后端**：
  - Python 3.8+
  - pip 20+
  - MySQL 8.0+
  - WSGI/ASGI 服务器（如 uvicorn）

### 4.2 部署流程

#### 4.2.1 前端部署

1. **安装依赖**：
   ```bash
   cd frontend
   npm install
   ```

2. **构建项目**：
   ```bash
   npm run build
   ```

3. **部署静态文件**：
   - 将 `dist` 目录部署到静态文件服务器
